# AI Sudoku Solver — React Frontend

A premium futuristic Sudoku solver interface built with React and Vite.

## Features

- Dark glassmorphism and neon-inspired dashboard design
- Animated Sudoku grid with glowing active cell effect
- Real-time backtracking visualization
- Difficulty selector and theme toggle
- Performance analytics and solver status display
- Keyboard input support
- Modular React component architecture

## Setup

1. Open a terminal in `react-ui`.
2. Run `npm install`.
3. Start the app with `npm run dev`.
4. Open the local URL shown by Vite.

## Notes

This React frontend is designed to run independently from the existing Python backend in the workspace. It includes a built-in backtracking solver for interactive visualization.
